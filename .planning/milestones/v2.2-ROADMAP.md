## Key Definitions

| Term | Definition |
|------|------------|
| Exit Trigger | Meticulous stop condition (weight, time, pressure, flow) |
| Exit Target | Gaggimate equivalent stop condition |
| Transition Mode | How the profile moves between phases (SMART, PRESERVE, LINEAR, INSTANT) |
| Relative Trigger | Meticulous +value syntax that accumulates across phases |
| Absolute Trigger | Gaggimate fixed-value trigger |
